- 파일 업로드

### 01. 메인화면

1. 기존과 같음
    
    → 단, 게시글 작성 일자 보이기 추가 (YYMMDD)
    
2. 프로필 사진 추가 (파일 업로드)
    - Nav에도 프로필 사진이 보임
    - 프로필 사진 클릭시 회원 정보 페이지 이동(마이페이지)
    
3. 로그인 / 로그아웃
    - 로그인 시 → NAV에 로그아웃 & 프로필사진 보임
    - 로그아웃 시 → NAV에 로그인 & 회원가입 보임
    
4. 회원가입
    - 회원가입시 프로필 이미지는 자동으로 기본 이미지로 대체됨
        - 회원정보 페이지에서 수정 가능

### 02. 회원정보페이지

1. 기본사항
    - 프로필 사진
    - 이름
    - 이메일
    - 비밀번호
    - 가입일 → 수정불가
    - 회원정보수정일 → 수정불가
    - 성별
    - 버튼 : 수정 / 탈퇴 / 취소

### 03. 회원정보 수정 페이지

1. 기본사항
    - 프로필 사진 → 이미지인데 버튼임ㅋㅋ 누르면 파일 수정 가능
    - 이름
    - 이메일
    - 비밀번호
    - 가입일 → 수정불가
    - 성별
    - 버튼 : 저장 / 취소

### 04. 게시글 상세 페이지

1. 기본사항
    - 제목
    - 작성자
    - 내용
    - 댓글
    - 버튼 : 수정(작성자만 입장 가능) / 삭제(작성자만 삭제 가능) / 취소

### 05. 댓글

1. 필요 사항
    - 댓글 작성 창 + 입력 버튼 필요
    - 작성된 댓글에 들어가야 하는 것
    - 프로필사진  / 댓글 작성자 이름 / 댓글 내용 / 댓글작성시간 / 본인 댓글 삭제&수정 기능
    - 수정/삭제
    - 모두에게 보이지만 권한은 자신한테만 있음
- 삭제 → 완전 삭제되서 안보임
